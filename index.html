HMC-STT 설치 가이드

  1. Chrome 브라우저에 chrome://extensions를 입력하여 확장 프로그램에 들어간다.

  2. dist.crx 파일을 끌어다 놓으면, 다음과 같은 팝업이 뜨고 "앱 추가" 버튼을 누른다.

  3. "앱 추가" 버튼을 누르면 HMC-STT 프로그램이 실행되고, 브라우저에서는 다음과 같이 설치된 앱 목록이 나온다.

  4. "HMC-STT"라고 되어있는 앱을 우클릭하여 "바로가기 만들기"를 선택한다.

  5. "애플리케이션 바로가기 만들기" 팝업이 뜨면 "만들기"를 선택한다.

  6. 이제 브라우저를 끄면 바탕화면에 "HMC-STT" 실행파일을 볼 수 있다.
